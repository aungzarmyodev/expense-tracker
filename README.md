# 📱 Expense Tracker App (Flutter)

A simple and secure Expense Tracker application built with Flutter.  
It allows users to manage daily expenses with **local storage (SQLite)**, **AES encryption**, and **multi-language support (English & Myanmar)**.

---

## ✨ Features

- ➕ Add new expenses (category, amount, date)
- 📋 View all expenses in a list
- 💾 Local storage using SQLite
- 🔐 AES encryption for sensitive data
- 🌍 Multi-language support (English & Myanmar)
- ⚡ State management using BLoC pattern
- 🎨 Clean and simple UI

---

## 🛠 Tech Stack

- Flutter
- Dart
- BLoC (`flutter_bloc`)
- SQLite (`sqflite`)
- AES Encryption (`encrypt`)
- Provider (Language state management)
- Intl (Localization support)
